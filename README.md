# sjobs
SoJob – Conectando Pessoas a Oportunidades  O SoJob é uma plataforma digital inovadora criada para facilitar o acesso a oportunidades de trabalho e serviços de forma rápida, simples e eficiente. O seu principal objetivo é conectar pessoas que procuram emprego com aquelas que oferecem serviços ou precisam de mão de obra, tudo em um único espaço.  .
